Getting Started

How To Start :

Step 1 - Unzip The Template File.

Step 2 - Install Code Editor Like Brackets Or Sublime Text.

Step 3 - Open The Project In The Code Editor.

Step 4 - Open Index.html on your web browser to see your website

Installation

How To Install Eclipse Ico Via FTP :

Step 1 - For faster load times it is recommended to minify your files after you customize them.
You can minify your css files Here
Minify html file Here
Minify Javascript files Here
Optimize Your images Here

Step 2 - Log into your hosting space via an FTP software (Filezilla for example).

Step 3 - Unzip your Download Pack file and open the extracted PURE template folder.

Step 4 - Upload the HTML file corresponding to the variant of your choice along with all directories (css,js,php,img) in the structure that they appear in your download package. Then, the HTML file should reside in the root directory, and all other files should reside inside their associated folder.

CSS

Inside the css folder There is four main style files styles-1-light.css , styles-1-dark.css , styles-2-light.css and styles-2-dark.css and a plugins.css which has all the css for all the plugins included

- styles-1-light.css is used in index-style-1.html and index-style-1-dark.html

- styles-1-dark.css is used in index-style-1.html and index-style-1-dark.html

- styles-2-light.css for all other light styles html files including blog pages, sign in and out pages and 404 page

- styles-2-dark.css is used for all the other dark styles files including blog pages, sign in and out pages and 404 page

Also you can find the sass files inside /css/sass folder

Fonts

This template uses google fonts, you can change the fonts for the paragraph or the headings

Step 1 - Go to google fonts Here and find a font you like.

Step 2 - Change lines 38 for paragraphs and line 48 for the headers in the styles-1.css OR styles-2.css files

Forms

There is two forms on the website, subscribe form and contact form

Subscribe Form

Subscription Form is handled by MailChimp, Please sign up Here, There is a free plan

Contact Form

We created two different methods to get your customers' emails and messages 1 through your server 2 through your gmail
Because some customers find the server method is hard and some customers don't have a gmail

Server Method
Step 1 - To use the server method add its script at the end of the index file and make sure you only have one contact script file NOT BOTH server and gmail script files

Step 2 - Change the Email Address to receive the email on and the message to receive as highlighted in the second image in contact-form-server.php file inside php folder

Gmail Method
Step 1 - To use the gmail method add its script at the end of the index file and make sure you only have one contact script file NOT BOTH server and gmail script files

Step 2 - Change the Gmail Address to receive the email on and your Gmail Password as highlighted in the second image in rd-mailform.config.json file inside php/bat folder

Step 2 - Change the message to receive as highlighted in the third image in contact-form-gmail.php file inside php folder

Some hosting companies like godaddy block some Mail ports so it is possible that both methods doesn't work for you
so we recommend using 123formbuilder Website And create a form with all the files you want and replace our form with the code you get

Step 3 - Change your company's exact position longitude and latitude in maps.js file in line 186 for the pin position on map.

Step 4 - Change your company's Address in maps.js file in line 193.
